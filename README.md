# .github

This repository contains the organization profile for **GitGuard**.

The profile appears on the organization page at https://github.com/yourusername *(update with your actual GitHub organization URL)*

## Structure

- `profile/README.md` - Organization profile content (displayed on GitHub organization page)
- `gitguard-mascot.png` - GitGuard mascot image used in the profile

## Note

GitHub requires organization profiles to be located at `profile/README.md` in a `.github` repository.

## About GitGuard

GitGuard is a comprehensive GitHub security scanning platform that helps developers identify vulnerabilities, secrets, and security risks in their repositories. Built with Next.js, TypeScript, and PostgreSQL, it provides enterprise-grade security scanning with AI-powered analysis.
